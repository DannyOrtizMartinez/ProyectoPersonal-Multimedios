# Referencias

## Documentación Oficial
* **Vue 3**: [https://vuejs.org/guide/introduction.html](https://vuejs.org/guide/introduction.html) - Referencia principal para la Composition API, reactividad, eventos y directivas.
* **MDN Web Docs**: [https://developer.mozilla.org/es/](https://developer.mozilla.org/es/) - Consulta de la API nativa `fetch`, manipulación del DOM, eventos de teclado y CSS Grid/Flexbox.

## Uso de Inteligencia Artificial
Durante el desarrollo de este minijuego, utilicé herramientas de IA generativa como apoyo para entender la estructura de la Composition API (`<script setup>`), interpretar y refactorizar la lógica de eventos de teclado, e implementar la reactividad y el binding de estilos de manera óptima. Me aseguré de revisar, probar y comprender cada línea de código sugerida antes de integrarla definitivamente en el proyecto, adaptando las soluciones a mis requerimientos específicos.
